# Music library

Drop royalty-free MP3 tracks here using these exact filenames:

- `mysterious.mp3`  — cosmic, suspenseful (astrology / spiritual)
- `romantic.mp3`    — warm, intimate (love / soulmate)
- `exciting.mp3`    — upbeat, energetic
- `calm.mp3`        — peaceful, meditative
- `dramatic.mp3`    — cinematic build
- `uplifting.mp3`   — bright, positive
- `epic.mp3`        — big, sweeping

The V0 wizard's Step 3 picker will show each track if the file exists. Missing
files appear greyed-out so it's obvious which moods you still need.
